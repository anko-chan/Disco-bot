# ぼくがかんがえたさいきょうのDiscord bot
欲しい機能を思いつくがままに四苦八苦しながら追加していきます。

### 現在の機能
- 雑多なシンプルな動作のGeneral commands
  - Admin限定のシャットダウン等のbot制御コマンド
  - xDy 形式のdice roll
- 某14番目のMMORPGに関した機能
  - パーティ募集ごっこ
  - クソマクロコンバーター(マクロのポジションシャッフル)
- 音声機能(全体的にwip)
  - mp3ファイルを再生するSoundboard

### 動作環境
- Python version 3.8.3
- Discord.py 1.6.0a
